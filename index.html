Hecho - API

Les endpoints suivants permettent de récupérer les données de l'API.

User

  • [GET] Get all users : /users
    ➔ https://hecho-api.onrender.com/users
  • [GET] Get an user with id : /user/{id}
    ➔ https://hecho-api.onrender.com/user/1

Authentification

  • [POST] Create a new user : /signup
    ➔ https://hecho-api.onrender.com/signup
  • [POST] Sign-in : /signin
    ➔ https://hecho-api.onrender.com/signin

Workout

  • [GET] Get all workouts : /workouts
    ➔ https://hecho-api.onrender.com/workouts
  • [GET] Get a workout with id : /workout/{id}
    ➔ https://hecho-api.onrender.com/workout/1
  • [POST] Create a new workout : /workout
    ➔ https://hecho-api.onrender.com/workout
  • [DELETE] Delete a workout : /workout/{id}
    ➔ https://hecho-api.onrender.com/workout/1
  • [PATCH] Update a workout : /workout/{id}
    ➔ https://hecho-api.onrender.com/workout/1
